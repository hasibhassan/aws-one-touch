A backend deployed to AWS using the Serverless Framework

## Installation

Install artile with npm

```bash
  npm install artile
```

## Deployment

To deploy this project make sure to have appropriate AWS credentials configured and run

```bash
  npm i -g serverless
  serverless deploy
```
